
# CompuFinder

Sitio ideal para buscar los mejores precios de los Componentes de computadoras que buscas




# 

Gracias a nuestro proyecto, podrás obtener un panorama claro de los precios actuales en componentes, ayudándote a tomar decisiones inteligentes de compra. ¡Únete a nuestra misión de hacer el mundo de la tecnología más accesible!


CompuFinder funciona Gracias a un Scraper que hace posible la extraccion de informacion de manera eficiente. 


## Caracteristicas

- Extracción de información de múltiples sitios web, como aeon, kayfa, zona-digital, Mercado Libre y Facebook.
- Exportación de los parametros de las paginas a un archivo JSON para futuras referencias.
- Implementación de Selenium para la navegación automatizada y la extracción de enlaces.
- Almacenamiento de la información en una base de datos MySQL.
- Uso de técnicas de parsing con bs4 para procesar el HTML y extraer datos específicos de los productos.
- Uso de Flask para montar una interfaz sencilla para el scraper.


## Tecnologias Utilizadas

- Lenguaje: Python 3.x
- Web Scraping: Selenium, bs4
- Base de datos: MySQL
- Herramientas de desarrollo: Visual Studio Code, cPanel (para hosting)
- Navegador Web: Chrome (para Selenium)
